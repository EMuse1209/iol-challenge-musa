# IOL-Challenge

Esta es mi solución para el Challenge de IOL, por Ezequiel Musa

## Instalación

* Ejecutar en una consola o terminal el comando: `npm install`.
* Para correr la solución: npm `start`.
## Uso

Click en el botón de más info. para obtener más información sobre el Pokemon elegido.
Puede utilizar el buscador para filtrar a los Pokemon por el nombre o parte del nombre ingresado en el campo.
### Comentarios Adicionales

La solución es 100% responsive. Fue creada con componentes de Material-UI, y los llamados a las APIs con Axios.
Hay mucha inconsistencia en el código (como distintas formas de utilizar estilos, de utilizar funciones de ES6+, etc.),
esto se hizo a propósito para demostrar el manejo de distintas técnicas y conocimientos.
